## File structure 

```
<terraform.zip>
  ├─ application/
  |   └─ main.tf
  |   └─ variables.tf
  ├─ main.tf
  └─ variables.tf
```
The main terraform file contains all information about 
providers and modules, whereas the main file inside 
the application directory contains all information about 
services.
